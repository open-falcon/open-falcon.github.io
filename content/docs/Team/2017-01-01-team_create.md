* users: 属於该群组的user id list
* resume: team的描述

### Request
```{"team_name": "ateamname","resume": "i'm descript", "users": [1]}```

### Response

```Status: 200```
```{"message":"team created! Afftect row: 1, Affect refer: 1"}```
